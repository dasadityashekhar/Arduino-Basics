# 3D LED Matrix Cube 💡

A 4x4x4 LED Cube built using Arduino to generate multiple 3D lighting animations and visual effects through LED multiplexing and layer control techniques.

This project was developed during my early learning phase in embedded systems and helped me understand Arduino programming, timing control, LED multiplexing, and hardware interfacing.

---

## 🚀 Features

- Multiple 3D LED animation patterns
- Layer-by-layer LED control
- Random rain lighting effect
- Spiral and propeller animations
- Flicker and wave effects
- Sequential LED traversal patterns
- Randomized LED effects

---

## 🛠️ Components Used

- Arduino UNO
- LEDs
- Resistors
- Jumper wires
- Breadboard / PCB
- Power supply

---

## ⚙️ Working Principle

The LED cube is controlled using:
- 16 column lines
- 4 layer lines

By rapidly switching layers and controlling columns, different LEDs appear to glow simultaneously, creating 3D lighting animations through multiplexing.

The Arduino controls:
- LED patterns
- Layer timing
- Animation sequencing
- Random visual effects
