# Arduino Basics 🔌

A collection of beginner and intermediate Arduino projects developed during my 1st and 2nd year while learning embedded systems, electronics, and microcontroller programming.

This repository contains small practical projects built using Arduino to understand sensors, displays, actuators, embedded logic, and hardware interfacing fundamentals.

These projects helped build the foundation for my later IoT and embedded systems projects.

---

## 📂 Projects Included

### 🔹 3D LED Matrix Cube
A multi-layer LED cube project demonstrating LED multiplexing and visual lighting effects using Arduino.

### 🔹 Calculator
A basic calculator implementation using Arduino and external input/output modules.

### 🔹 Scrolling Text Display Using Arduino
A text scrolling display project using LED matrix/display modules for message visualization.

### 🔹 Temperature Controlled Fan
An automatic fan speed/control system based on surrounding temperature sensor readings.
