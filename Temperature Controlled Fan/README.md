# Temperature Controlled Fan 🌡️

An automatic temperature-controlled fan system developed using ESP32 and a DHT11 temperature sensor.

The project automatically controls a cooling fan based on surrounding temperature conditions. This project was developed during Xamble 2.0, an electronics competition organized by the SIGNODES IoT Club.

This was my first participation in an electronics competition during the 2nd semester of my engineering journey, and our project secured 🥈 Second Prize.

## ⚙️ Working Principle

- The DHT11 sensor continuously measures surrounding temperature and humidity.
- ESP32 reads sensor data in real time.
- When temperature exceeds a predefined threshold:
  - The fan automatically turns ON.
- When temperature drops below the threshold:
  - The fan turns OFF.

This creates a simple automatic cooling system.
