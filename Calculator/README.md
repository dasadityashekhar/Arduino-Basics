# Arduino Calculator 🧮

A simple calculator built using Arduino, a 4x4 keypad, and a 16x2 LCD display to perform basic arithmetic operations.

This project was developed during my early learning phase in embedded systems and helped me understand keypad interfacing, LCD communication, user input handling, and embedded logic implementation.

---

## 🚀 Features

- Basic arithmetic operations
  - Addition
  - Subtraction
  - Multiplication
  - Division
- Real-time input display
- LCD-based result output
- Keypad-controlled operation
- Reset/Clear functionality

---

## 🛠️ Components Used

- Arduino UNO
- 4x4 Matrix Keypad
- 16x2 LCD Display
- Jumper wires
- Breadboard
- Resistors

---

## ⚙️ Working Principle

- The user enters numbers using the 4x4 keypad.
- Arithmetic operations are selected through dedicated keypad buttons.
- The Arduino processes the input and performs calculations.
- Results are displayed on the 16x2 LCD screen.

The project uses:
- Keypad library for matrix keypad handling
- LiquidCrystal library for LCD interfacing

---

## ⌨️ Keypad Layout

| Key | Function |
|---|---|
| `0-9` | Number Input |
| `+` | Addition |
| `-` | Subtraction |
| `*` | Multiplication |
| `/` | Division |
| `=` | Calculate Result |
| `C` | Clear/Reset |
