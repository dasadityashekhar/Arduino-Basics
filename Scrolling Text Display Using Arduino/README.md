# Scrolling Text Display Using Arduino 📟

An Arduino-based scrolling text display project developed using LED matrix modules and shift registers to display animated scrolling text.

This project helped me learn display interfacing, shift register communication, LED matrix control, and animation logic using Arduino.

---

## 🚀 Features

- Smooth scrolling text animation
- Custom text message display
- LED matrix multiplexing
- Shift register-based control
- Adjustable scrolling speed
- Character bitmap rendering

---

## 🛠️ Components Used

- Arduino UNO
- LED Matrix Display
- 74HC595 Shift Registers
- Jumper wires
- Breadboard
- Resistors

---

## ⚙️ Working Principle

The project uses:
- LED matrix display for text visualization
- 74HC595 shift registers for controlling rows and columns
- Bitmap-based font rendering for character generation

The Arduino:
1. Converts characters into bitmap patterns
2. Sends row and column data using shift registers
3. Continuously refreshes the display
4. Shifts bitmap data to create scrolling animation
